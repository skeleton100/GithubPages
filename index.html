ScoreBoard

麻雀点数計算アプリ

アプリについて

ScoreBoardは、麻雀の点数計算を簡単に記録・管理できるシンプルなアプリです。対局の得点を記録し、ウマオカを含めた最終結果を自動計算します。

✓ 簡単な点数入力
直感的なUIで素早く点数を記録できます
✓ 自動計算
ウマオカを含めた最終結果を自動で計算
✓ 履歴管理
過去の対局記録を保存・確認できます
✓ カスタマイズ可能
ルール設定（ウマ・オカ・順位点）を自由に変更可能

プライバシーポリシー

1. 収集する情報

本アプリは、以下の情報を収集・保存します：

  • アプリ内で入力された対局記録（プレイヤー名、得点など）
  • アプリの設定情報

これらの情報は、すべてお客様のデバイス内にローカルで保存されます。

2. 情報の利用目的

収集した情報は、以下の目的でのみ使用されます：

  • 麻雀の対局記録の保存と表示
  • 点数計算の実行
  • アプリの機能提供

3. 情報の共有と第三者提供

本アプリは、お客様の個人情報を第三者と共有することはありません。すべてのデータはデバイス内にのみ保存され、外部サーバーへの送信は一切行いません。

4. データの保存と削除

すべてのデータはデバイス内のローカルストレージに保存されます。アプリをアンインストールすることで、すべてのデータが削除されます。

5. セキュリティ

お客様のデータは、デバイスのセキュリティ機能によって保護されます。本アプリは、データを暗号化せずにローカルデータベースに保存します。

6. 分析ツール・広告

本アプリは、広告の配信やアクセス解析ツールを使用していません。

7. プライバシーポリシーの変更

本プライバシーポリシーは、必要に応じて変更される場合があります。変更後のプライバシーポリシーは、本ページに掲載された時点で効力を生じるものとします。

8. お問い合わせ

本プライバシーポリシーに関するお問い合わせは、以下のGitHubリポジトリのIssuesよりご連絡ください。
